Mastering Embedded Questions

Learn how word order changes when a question is embedded inside a statement or another question.

Rule 1: The "Un-Question Switch"

When a question is embedded, the word order switches from Verb-Subject to Subject-Verb.

Question Word where / what time
Target Verb is / are / can
→
Question Word where / what time
Subject the test / she

Rule 2: The Disappearing "Do/Does/Did"

Never use do/does/did in an embedded question! Drop the helper verb entirely and change the main action to match the tense.

Question Word where
Helper Verb did ❌
→
Question Word where
Main Action drove

"Be" Verbs

What time is the exam?
Do you know what time the exam is?

Modal Verbs

Where can I park?
Can you tell me where I can park?

Do / Does / Did

Where did he drive?
I don't know where he drove.
