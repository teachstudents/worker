Quick Reference: Questions with Some/Any & Much/Many

Some in Questions

  • Offers: Would you like some tea?
  • Requests: Can I have some water?
  • Expected "Yes": Do you have some time?
  • Pattern: some + noun (countable plural/uncountable)

Any in Questions

  • General questions: Do you have any questions?
  • Negative expectation: Is there any milk left?
  • With countable/uncountable: any books / any information
  • Pattern: any + noun (in most questions)

Much in Questions

  • Uncountable nouns only: How much time? How much homework?
  • Not for countable: ✗ How much books? ✓ How many books?
  • Pattern: How much + uncountable noun

Many in Questions

  • Countable plural nouns: How many students? How many apples?
  • Not for uncountable: ✗ How many water? ✓ How much water?
  • Pattern: How many + countable plural noun

Is there / Are there

  • Is there + any + uncountable: Is there any water?
  • Are there + any + countable plural: Are there any books?
  • Key: Verb agrees with noun, not with "any"

Common ZhongKao Errors

  • ✗ How much students? ✓ How many students?
  • ✗ Do you have some questions? ✓ Do you have any questions?
  • ✗ Is there any books? ✓ Are there any books?
  • Tip: Identify noun type first, then choose quantifier
